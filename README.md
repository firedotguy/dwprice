# dwprice
Сайт для простмора цен на вещи в магазинах городов DesireWorld (сервера по Minecraft)
https://discord.gg/VRj4Neff

### Зачем пользоваться?

#### Для покупателей
 - Видно среднюю цену товара и цены у разных магазинов (чтобы не взять у кого за дорого)
 - Видно стоимость товара (удобно при обменах)

#### Для магазинов
 - Привлечение новых покупателей (магазины с самыми низкими ценами отображаются выше всего)
 - Наблюдение за конкурентами
